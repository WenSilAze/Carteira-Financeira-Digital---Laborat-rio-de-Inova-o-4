# Carteira Digital Financeira

Uma aplicação desenvolvida em **PHP**, **JavaScript** e **SQLite** para **acompanhar ganhos e gastos** de forma prática.  
Você pode **alterar manualmente os valores** de depósitos e saques e visualizar um **gráfico interativo** que mostra a evolução dos seus projetos ao longo do tempo.

## Funcionalidades

- **Controle de ganhos e gastos** com edição manual  
- **Registro de depósitos e saques** por projeto  
- **Gráfico interativo** para acompanhar a evolução financeira  
- **Histórico organizado** de todas as movimentações  
- **Interface simples** para uso rápido e direto  

## Tecnologias Utilizadas

| Tecnologia | Função |
|------------|--------|
| **PHP** | Backend e lógica de negócio |
| **JavaScript** | Interatividade e renderização do gráfico |
| **SQLite** | Banco de dados leve para armazenar transações |

1. Cadastre um projeto (ex: "Reforma da casa").
